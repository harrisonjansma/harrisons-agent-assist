# Bulk Editing and Retargeting Links

Use this procedure when a creator needs to change many links at once — for example swapping an affiliate tag or retargeting a whole retailer that changed URLs.

## What bulk editing can do
- Select multiple Library items and apply one change: append or replace an affiliate tag, swap a domain, or move them to a collection.
- Changes preview before they apply so the creator can confirm the count.

## Steps
1. In **Library**, use the checkboxes (or **Select all** with a filter) to choose the items.
2. Open **Bulk actions → Retarget links** and choose the operation (e.g. replace `oldshop.com` with `newshop.com`).
3. Review the preview count and confirm — the change runs in the background and updates each card's link status.

## Common issues
- **Some items skipped:** their URLs didn't match the find pattern; narrow the filter and retry.
- **Wrong tag applied:** re-run with the correct tag — bulk edits are not versioned, so double-check before confirming.

## Escalate when
- A bulk operation reports success but the Library still shows old links after a refresh.
