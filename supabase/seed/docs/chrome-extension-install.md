# Installing the Chrome Extension

Use this procedure when a creator can't install the Shopfolio Chrome extension, or the **Capture** button doesn't appear while browsing.

## What the extension does
- The extension adds a **Capture** button to product pages so creators can pull an image, title, price, and link straight into their Library.
- It signs in with the creator's existing Shopfolio session — no separate password.

## Steps to install
1. Send the creator to the extension listing from **Settings → Chrome Extension**.
2. Have them click **Add to Chrome** and pin the extension via the puzzle-piece menu.
3. Click the icon once and confirm it shows their account email — this links it to their workspace.

## Common issues
- **Capture button missing:** the extension needs a page reload after install; refresh the product tab.
- **Shows "not signed in":** the creator is logged into Shopfolio in a different Chrome profile — match the profiles.

## Escalate when
- The extension installs but never authenticates despite a valid, active session.
