# Mobile App and PWA

Use this procedure when a creator wants Shopfolio on their phone, or the installed web app won't open, update, or stay signed in.

## How the mobile app works
- Shopfolio ships as a Progressive Web App (PWA) — there's no separate App Store or Play Store download. Creators install it from the browser.
- Once installed it runs full-screen with its own icon and keeps its session like the desktop site.

## Steps to install
1. **iOS Safari:** open the site, tap **Share**, then **Add to Home Screen**.
2. **Android Chrome:** open the site, tap the menu, then **Install app** (or accept the install prompt).
3. Launch it from the new home-screen icon and sign in once.

## Common issues
- **Stale version / blank screen:** the PWA cached an old build — have them close it fully and reopen, or reinstall to force an update.
- **Signed out constantly:** iOS clears storage for apps it thinks are unused; opening it regularly avoids this.

## Escalate when
- A reinstall still loads a blank screen after a recent deploy.
