# Two-Factor Authentication

Use this procedure when a creator wants to enable 2FA, is locked out by it, or lost their authenticator device.

## How 2FA works
- Shopfolio supports app-based (TOTP) 2FA — the creator scans a QR code into an authenticator app and enters a 6-digit code at sign-in.
- Enabling 2FA generates one-time **backup codes** to store safely.

## Steps to enable
1. In **Settings → Security**, click **Enable two-factor** and scan the QR code with an authenticator app.
2. Enter the current 6-digit code to confirm the pairing.
3. Save the backup codes shown — each works once if the device is unavailable.

## Locked out
- Have the creator use a **backup code** at the 2FA prompt, then re-pair a new device from Security.
- If backup codes are also lost, verify identity against account details before any reset.

## Escalate when
- The creator lost both the device and all backup codes (requires a verified identity reset).
