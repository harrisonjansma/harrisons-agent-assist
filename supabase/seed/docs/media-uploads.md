# Image and Media Uploads

Use this procedure when a creator can't upload an image, an upload fails, or media looks blurry or wrongly cropped.

## What we accept
- JPEG, PNG, and WebP up to 20 MB per image. Pinterest publishing favors a 2:3 (vertical) ratio.
- Uploads go to object storage and are served through Cloudflare's image CDN.

## Steps
1. In the composer or Library, click **Upload** and choose a file within the size and format limits.
2. Use the crop tool to set the 2:3 frame before saving.
3. Confirm the thumbnail renders — a broken thumbnail means the upload didn't complete.

## Common issues
- **Upload stalls or fails:** usually an oversized file or a flaky connection; re-save under 20 MB.
- **Blurry pin:** the source image is low-resolution — advise at least 1000 px on the short edge.
- **Wrong crop on Pinterest:** re-crop to 2:3, since Pinterest trims wide images.

## Escalate when
- Uploads fail for a creator across formats and sizes despite a stable connection.
