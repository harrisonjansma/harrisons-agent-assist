# Customizing Your Storefront

Use this procedure when a creator asks how to change the look of their link-in-bio page or pop-up shop, or how to embed it.

## What's customizable
The storefront is a customizable, embeddable link-in-bio / pop-up shop. Creators can set the theme (colors, fonts, layout), reorder products and collections, add a bio and social links, and toggle between the full page and an embeddable widget.

## Common tasks
1. **Change the look:** **Storefront → Design** — pick a theme and adjust colors/fonts; changes preview live and publish on save.
2. **Reorder items:** drag products or whole collections on **Storefront → Layout**.
3. **Embed it elsewhere:** copy the embed snippet from **Storefront → Embed** and paste it into another site; it stays in sync with the hosted page.

## Serving on a custom domain
To serve the storefront on the creator's own domain, see *Connecting a Custom Domain*. Design and layout carry over automatically.

## Escalate when
- An embed won't render inside a specific host that strips iframes/scripts.
